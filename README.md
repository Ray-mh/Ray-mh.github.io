# Ray's Wonderland - Portfolio Website

This is the complete, deployment-ready package for Ray's Wonderland portfolio website.

## 📦 Package Contents

- `index.html` - Home page with project gallery and category filtering
- `projects.html` - Complete portfolio showcase
- `experiences.html` - Work history, certifications, and education
- `skills.html` - Skills and competencies
- `about.html` - Personal introduction
- `contact.html` - Contact information and form
- `css/` - Complete stylesheet with responsive design
- `js/` - Interactive features (menu, filtering, forms)
- `images/` - All project images and assets

## 🚀 Deploying to GitHub Pages

### For Ray-mh.github.io repository:

1. **Upload these files to your repository:**
   - Go to https://github.com/mohammadi-hadi/Ray-mh.github.io
   - Upload all files and folders from this package to the **root** of the repository

2. **Enable GitHub Pages:**
   - Go to repository Settings → Pages
   - Under "Source", select branch: `main`
   - Select folder: `/ (root)`
   - Click Save

3. **Your site will be live at:**
   - https://mohammadi-hadi.github.io/Ray-mh.github.io/
   - Or if you configure a custom domain: your-custom-domain.com

### Alternative: Create a new repository

1. Create a new repository named `username.github.io` (replace username with your GitHub username)
2. Upload all these files to the repository root
3. Your site will automatically be published at `https://username.github.io`

## 🛠️ Customization

### Updating Content:
- Edit the HTML files directly to update text content
- Replace images in the `images/` folder (keep same filenames)
- Update styles in `css/style.css`

### Adding Projects:
- Add images to appropriate `images/` subfolder
- Update `projects.html` with new project information
- Update `index.html` gallery if needed

### Changing Colors:
- Edit `css/style.css`
- Search for color codes (e.g., `#D9FFD4`) and replace
